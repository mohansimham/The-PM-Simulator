An interactive executive simulation for testing product management skills based on real product challenges.

## Quick Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from this folder
3. Go to Settings ‚Üí Pages ‚Üí Source: GitHub Actions
4. Your site will be live in 2-3 minutes!

## Files Included

- `index.html` - Main HTML file
- `pm-simulation.jsx` - React simulation component
- `.github/workflows/deploy.yml` - Automatic deployment
- `GITHUB_DEPLOYMENT_GUIDE.md` - Detailed setup instructions

## Google Sheets Integration

Your simulation is already connected to your Google Sheet:
`https://script.google.com/macros/s/AKfycbz6E8UJUn5xhilm2pLWjdvaip_6MnSCf-XRVPv5u4hOttjr0SNCu9I10q2ESLKGQ04_mQ/exec`

## Customization

Want to change the simulation content?
- Edit questions in `pm-simulation.jsx`
- Search for `simulation` object (around line 35)
- Modify scenarios, options, and scoring
